Creation Form Help

Here you have the option to select either the Kubernetes Cluster or the GPU. To navigate this form, you first make your selection, then click on the buttons to the left to travel through the form.
If you want to stop filling out the Cluster Form or vice versa, just go back to the selection tab and select the other option. It will clear your old form and allow you to navigate your desired form.
After making sure to fill out all of the input fields, you will be able to then submit the form at the review page, which shows what you have inputted.
Here are some quick explanations as to what certain parts of the forms refer to:

Type of Server (Machine Type): https://docs.microsoft.com/en-us/azure/cloud-services/cloud-services-sizes-specs
Location: Selection from az account list-locations
Machine Type: Type of Server (n1-standard-2 for cheap alternative/ otherwise go for n1-highmem-4)
Storage Account, Container Name, Storage Name, and Storage Name: For use with Pachyderm
Number of Nodes: Default number of nodes
Number of Nodes (Class): The number of nodes when class is in session
